函数和方法

Golang函数和方法的用法及区别

# 1. 函数

# 1.1 函数定义

func 函数名(参数列表) (返回值列表) {
    // 函数体
}

代码：

package main

import(
    "fmt"
)


func Add(i int, j int) (int) {  //常规函数
    return i+j
}
func Add_Multi_Sub(i, j int) (int, int, int) {  //多值返回
    return i+j, i*j, i-j
}

func sum(nums ...int) { //变参函数
    total := 0
    for _, num := range nums {
        total += num
    }
    fmt.Println(total)
}
func main(){
    a, b := 2,3
    arr := []int{1, 2, 3}
    var c int = Add(a,b)
    d,e,f := Add_Multi_Sub(a,b)
    fmt.Println(c,d,e,f)
    sum(arr...)  //注意传参形式

}

# 2. 方法

Go语言中的方法和函数很相似，本质上是作用在接受者上的函数，接受者是非内置类型的变量（可以是内置类型的别名），使用方法和java中类的方法一样。

使用方法可以很巧妙的实现java中的类，只不过Go语言里类型和方法耦合更宽松，方法没有和数据定义（结构体）混在一起：它们是正交的类型；表示（数据）和行为（方法）是独立的。

# 2.1 定义和调用

type myInt int //给内置类型定义别名

// 定义
func (a myInt) Add (b myInt){
    fmt.Println(a + a)
}

// 使用
// 表达式a.Add称作选择子（selector），它为接收者a选择合适的Add方法
var a,b myInt = 3,4
a.Add(b)

# 3. 函数和方法的区别

  • 方法是特殊的函数，定义在某一特定的类型上，通过类型的实例来进行调用，这个实例被叫接收者(receiver)。
  • 函数将变量作为参数：Function1(recv)
  • 方法在变量上被调用：recv.Method1()
  • 接收者必须有一个显式的名字，这个名字必须在方法中被使用。
  • receiver_type 叫做 （接收者）基本类型，这个类型必须在和方法同样的包中被声明。
