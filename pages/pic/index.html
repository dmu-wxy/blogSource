树状数组&线段树

参考：博客园@Xenny (opens new window)

# 1.什么是树状数组&线段树

都是用数组来模拟树形结构。

为什么不直接建树？

没必要，能用树状数组处理的问题都没必要建树。

# 2.能解决什么问题

可以解决大部分基于区间上的更新以及求和问题。

# 3.树状数组和线段树的区别

树状数组可以解决的问题都可以用线段树解决。但是树状数组的系数要比线段树少很多。

# 4.树状数组的优缺点

优点： 修改和查询的复杂度都是O(logN)，而且相比线段树系数要少很多，比传统数组要快，而且容易写。

缺点： 不能解决复杂的区间问题，功能有限。

# 5.线段树&树状数组

线段树：

​ 父节点的值为子节点值的和，子节点二分父节点。

树状数组：

​ 父节点的值为子节点值的和，子节点按一定规则分父节点的值。

其规律为

c[i] = a[i - 2^k + 1] + a[i - 2 ^ k + 2] +…… +a[i]; (c为图中红色节点，a为图中黑色节点，k为i二进制中从低位到高位连续0的长度)

import java.util.*;

public class Main {
	private static int n;
	private static int m;
	private static int[] nums;
	public static void main(String[] args) {
		Scanner input = new Scanner(System.in);
		n = input.nextInt();
		m = input.nextInt();
		nums = new int[n + 5];
		for(int i = 1;i <= n;i++) {
			int temp = input.nextInt();
			add(i,temp);
		}
		for(int i = 0;i < m;i++) {
			int choice = input.nextInt();
			int x = input.nextInt();
			int y = input.nextInt();
			if(choice == 1) {
				add(x,y);
			}else {
				System.out.println(sum(y) - sum(x - 1));
			}
		}
		input.close();
	}
	public static int sum(int x) {
		int ans = 0;
		while(x != 0) {
			ans += nums[x];
			x -= lowbit(x);
		}
		return ans;
	}
	public static void add(int x,int k) {
		while(x <= n) {
			nums[x] += k;
			x += lowbit(x);
		}
	}
    //求k
	public static int lowbit(int x) {
		return x & -x;
	}
}
