# 颜色

我们所能看到的物体的颜色由**物体本身的颜色**与**光源发出的颜色**的结合而成，在代码中反应为：

```cpp
glm::vec3 lightColor(0.33f,0.42f,0.18f);  // 光源颜色
glm::vec3 toyColor(1.0f,0.5f,0.31f);  // 物体颜色
glm::vec3 result = lightColor * toyColorl; // = (0.33f,0.21f,0.06f)
```
